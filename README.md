### 📄 Context-Aware PDF Chatbot using LangChain

Developed a Retrieval-Augmented Generation (RAG) chatbot that allows users to upload PDF documents and ask questions using natural language. Implemented document loading, text chunking, vector embeddings, semantic retrieval, and context-aware response generation.

**Tech Stack:** Python • LangChain • Google Gemini • Streamlit • InMemoryVectorStore • RAG

**Tech Stack:** Python • Scikit-learn • Pandas • NumPy • PCA • UMAP • t-SNE
